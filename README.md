# sortingVisulizer
It is a fun tool that visualizes famous sorting Algorithms:
* Bubble Sort
* Selection Sort
* Insertion Sort
* Quick Sort
* Merge Sort
With this tool we can see yhe working of these algorithms with bar graph.
